# Affordable Dwelling Unit Documents

This folder currently contains a provisional affordable-unit program document.

It should not be treated as a complete or clean Affordable Dwelling Unit Ordinance, WDU, or MPDU class source. It is useful for early smoke testing because it contains AMI restrictions, income-qualified affordable units, developer/program participation, and affordability requirements.
